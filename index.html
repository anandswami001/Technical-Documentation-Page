HTML Basics

HTML stands for HyperText Markup Language, and it's the standard markup language for creating web pages.

It allows us to structure content on the web by using a system of elements, or tags, to define different parts of a webpage.

HTML documents consist of nested elements that represent different parts of the page.

<!DOCTYPE html> <html> <head> <body> <h1>Hello World</h1>
  • HTML uses tags to define page structure.
  • Common tags include <div>, <span>, <a>, and <img>.
  • The <head> section contains metadata about the page, such as title and links to external files like CSS.
  • The <body> section contains the actual content of the page.
  • HTML elements can have attributes like class, id, href, and src for additional functionality.
CSS Styling

CSS (Cascading Style Sheets) is used to control the presentation, formatting, and layout of a web page.

It enables developers to apply different styles to HTML elements, such as colors, fonts, spacing, and alignment.

CSS can be included directly in the HTML file, within a <style> tag, or externally using a <link> tag.

body {} h1 {color: blue;} .container { display: flex; } p {font-size: 14px;} img {width: 100%;}
  • CSS uses selectors to target elements based on their tag, class, id, or attributes.
  • Styles can be applied to individual elements, groups of elements, or based on user interaction (hover, focus, etc.).
  • External CSS files help keep code modular and maintainable.
  • CSS frameworks like Bootstrap and Tailwind help speed up the styling process.
  • Media queries allow you to create responsive designs for different screen sizes.
JavaScript Fundamentals

JavaScript is a high-level programming language that enables interactive web pages. It is an essential part of the modern web development stack.

With JavaScript, you can manipulate HTML and CSS, handle events, and perform asynchronous operations like fetching data from a server.

JavaScript code can be added directly to an HTML file within <script> tags or in external files.

let x = 5; const y = 10; document.querySelector('h1').textContent = 'Hello JavaScript'; fetch('url').then(response => response.json()); function add(a, b) { return a + b; }
  • JavaScript uses variables, functions, and objects to structure data and behavior.
  • DOM manipulation allows you to change HTML and CSS dynamically.
  • JavaScript supports both synchronous and asynchronous code execution.
  • Common asynchronous operations include HTTP requests, event handling, and setTimeout.
  • JavaScript frameworks and libraries like React, Angular, and Vue.js simplify complex tasks.
Git and GitHub

Git is a version control system that helps developers track changes in code. GitHub is a platform that hosts Git repositories, allowing teams to collaborate on projects.

With Git, you can commit changes, create branches, and merge code. GitHub enables easy collaboration with features like pull requests, issue tracking, and code reviews.

git init git add . git commit -m "Initial commit" git push origin main git pull origin main
  • Git stores the history of changes in a repository, allowing you to revert to previous versions.
  • Git branches allow developers to work on new features without affecting the main project.
  • GitHub allows developers to store their Git repositories online and share them with collaborators.
  • GitHub provides tools for code review, such as pull requests and inline comments.
  • Git commands can be used in a terminal or through GUI tools like GitHub Desktop.
Web Development Tools

Web developers use a wide range of tools to write, test, and deploy code efficiently. These tools help with coding, debugging, version control, and collaboration.

Some essential web development tools include code editors, browser developer tools, and task runners.

VS Code Chrome Developer Tools Git Jest Webpack
  • VS Code is a popular text editor with built-in features like syntax highlighting, extensions, and debugging.
  • Chrome Developer Tools allow you to inspect and debug web pages in real-time.
  • Git is essential for tracking code changes and collaborating with others.
  • Jest is a testing framework for writing unit tests and verifying that code works as expected.
  • Webpack is a module bundler that helps optimize and bundle JavaScript, CSS, and other assets for production.
